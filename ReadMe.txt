GuruPreneur adalah platform yang menghubungkan murid dengan mentor/tutor secara online.
Murid dapat mencari dan memesan mentor sesuai kebutuhan belajar mereka.

- Registrasi dan login untuk mentor & murid
- Murid dapat mencari mentor berdasarkan kategori skill
- Mentor dapat mengatur portofolionya
- Sistem hire/pemesanan mentor oleh murid

- Next.js 14
- Prisma ORM
- PostgreSQL
- Tailwind CSS
- shadcn

< Cara Menjalankan Project >
git clone <repo-url>
cd GuruPreneur-web
npm install
npm run dev

Student:
- Melihat dan hire mentor

Mentor:
- Mengatur profil & portofolio
- Melihat data mahasiswa yang hire
